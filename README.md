Counter App

A simple and interactive counter application built using HTML, CSS, and JavaScript. The app allows users to increase, decrease, and reset a numeric value dynamically.

---

Features

* Increase the counter value
* Decrease the counter value
* Reset the counter to zero
* Instant UI updates using JavaScript
* Clean and minimal user interface

---

Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

How It Works

* A variable `count` is initialized to 0
* Three buttons control the counter:

  * **Increase** → increments the value
  * **Decrease** → decrements the value
  * **Reset** → resets value to 0
* The updated value is displayed using `textContent`
